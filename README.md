# P'YUI GPT Remake

P'YUI GPT Remake คือโปรเจกต์พัฒนาแชทบอทแบบ RAG ขึ้นใหม่ โดยใช้ระบบของรุ่นพี่เป็นกรณีศึกษาและแหล่งอ้างอิงในการทำความเข้าใจโครงสร้างระบบ แต่ไม่ได้คัดลอกโค้ดหรือพัฒนาต่อจากระบบเดิมโดยตรง

โปรเจกต์นี้มีเป้าหมายเพื่อศึกษาองค์ประกอบของระบบแชทบอทแบบแยกส่วน เช่น การนำเข้าข้อมูล การประมวลผลเอกสาร การสร้าง Embedding การจัดเก็บข้อมูลใน Vector Database การค้นคืนข้อมูล การใช้โมเดลภาษาในการสร้างคำตอบ และการออกแบบ Workflow ของระบบ

ฐานข้อมูลหรือชุดข้อมูลเดิมจากโปรเจกต์ของรุ่นพี่สามารถนำมาใช้เป็นข้อมูลตั้งต้นสำหรับการทดลองระบบได้ แต่โครงสร้างระบบ กระบวนการทำงาน และส่วนเสริมต่าง ๆ จะถูกออกแบบและพัฒนาขึ้นใหม่

## เป้าหมายของโปรเจกต์

- ศึกษาโครงสร้างของระบบ RAG Chatbot ตั้งแต่พื้นฐาน
- แยกองค์ประกอบของระบบออกเป็นส่วนย่อยเพื่อทำความเข้าใจทีละส่วน
- เปรียบเทียบเทคโนโลยีเดิมที่รุ่นพี่ใช้กับทางเลือกใหม่
- เลือกใช้เครื่องมือหรือโมเดลที่เหมาะสม โดยเน้น Open Source หรือ Open Model
- พัฒนาระบบใหม่ที่สามารถนำฐานข้อมูลเดิมมาใช้ได้
- เพิ่มหน้าจอสำหรับผู้ดูแลระบบในการเพิ่มและจัดการฐานความรู้
- ศึกษาความเป็นไปได้ในการใช้ LangGraph แทน LangChain หากเหมาะสม
- เพิ่มระบบประเมินคำตอบและระบบบันทึกการทำงานของแชทบอท

## แนวทางการพัฒนา

การพัฒนาจะเริ่มจากการศึกษาแต่ละส่วนของระบบแยกกันก่อน จากนั้นจึงค่อยนำแต่ละส่วนมาประกอบรวมกันเป็นระบบแชทบอทที่สมบูรณ์
